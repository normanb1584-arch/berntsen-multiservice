# Berntsen Multiservice

Nettside for Berntsen Multiservice bygget med Next.js og Tailwind CSS.

## Komme i gang

Installer avhengigheter:

```bash
npm install
```

Kjør utviklingsserveren:

```bash
npm run dev
```

Åpne [http://localhost:3000](http://localhost:3000) i nettleseren for å se resultatet.

## Bygge for produksjon

```bash
npm run build
npm start
```

## Teknologier

- **Next.js 14** - React-rammeverk med App Router
- **TypeScript** - Type-sikkerhet
- **Tailwind CSS** - Utility-first CSS-rammeverk
- **React 18** - UI-bibliotek
